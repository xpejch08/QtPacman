\doxysubsection{Detailed Description}
Represents the game board log in a Pac-\/\+Man game. 

The \mbox{\hyperlink{classGridLog}{Grid\+Log}} class is a QGraphics\+Item that represents the game board log in a Pac-\/\+Man game. It is responsible for reading the log files and drawing the log messages on the screen.
