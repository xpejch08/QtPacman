\doxysubsection{Detailed Description}
Represents the view of a ghost character in a Pac-\/\+Man game. 

The \mbox{\hyperlink{classGhost}{Ghost}} class is a subclass of QObject and QGraphics\+Pixmap\+Item, and represents a ghost character in a Pac-\/\+Man game. It is responsible for moving around the game board and avoiding collisions with the player character and other ghosts.
