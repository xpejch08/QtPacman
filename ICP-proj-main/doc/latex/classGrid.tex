\doxysubsection{Detailed Description}
Represents the game board in a Pac-\/\+Man game. 

The \mbox{\hyperlink{classGrid}{Grid}} class is a QGraphics\+Item that represents the game board in a Pac-\/\+Man game. It is responsible for drawing the maze and the player and ghost characters on the screen.
